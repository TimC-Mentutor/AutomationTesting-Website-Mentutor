## How To Run Project.
1. Clone use link below
```
$ git clone https://github.com/TimC-Mentutor/AutomationTesting-API-Mentutor.git
```
3. Go to directory
```
$ cd API-Mentutor
```
5. Run the project
```
$ mvn clean verify
```
7. You good to go!
